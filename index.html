Tile game demo (work in progress!)

Tile based 2D game demo written in C++ using SDL2 for cross-platform windowing and input, and OpenGL for drawing.
I wanted to build a game that used tile renderering while also practicing my C++ skills. Another thing that I wanted to do was experiment with Lua integration.

Features so far...

  • Basic Lua integration - Certain functions (like enemy movement and attacks) are exposed to Lua using the Lua C API
  • Efficient tile map rendering
  • Procedural dungeon generation using binary space partitioning
  • Tile-based dynamic lighting using a Dijsktra map
